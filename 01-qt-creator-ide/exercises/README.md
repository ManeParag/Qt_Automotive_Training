# Module 01 — Exercise

Copy the starter project from `../HelloAutoHMI/` into a new folder
here named `HelloAutoHMI-<your-name>/` and complete the tasks below.

## Tasks

1. Add a second button labeled **Headlights** that toggles between *On* and *Off* and reflects in the status bar.
2. Add a third button **Reset** that returns engine and headlights to their default off state.
3. Make the title label red when the engine is running and grey when it's off (use `setStyleSheet`).

## Submission checklist

- [ ] Code builds clean with no warnings (`make` or *Build* in Qt Creator)
- [ ] App launches without crashing
- [ ] All tasks above implemented
- [ ] Screenshot of the running app added as `screenshot.png` in your folder
- [ ] Branch named `module-01/<your-name>`, PR opened against `main`
